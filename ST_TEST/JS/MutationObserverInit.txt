# MutationObserver

    new MutationObserver()
        callback callback> 观察到改变后回调改函数，该函数会被传递mutationLists: MutationRecord对象，包含被改变的元素，改变方式等属性;observer: 观察到节点改变的观察者实例
        + 创建一个观察者实例

## observer实例方法
    observe
        targetNode DOM> 被监听的节点
        config Object> observerConfig监听配置
        + 使用配置对目标节点进行观测

    disconnect
        + 停止观测目标节点

### observerConfig

    attributeFilter
        + 可选， 要监视的特定属性名称的数组。如果未包含此属性，则对所有属性的更改都会触发变动通知。无默认值

    attributeOldValue
        + 可选，当监视节点的属性改动时，将此属性设为true将记录任何有改动的属性的上一个值。 无默认值

    attributes
        + 可选，设为true将以观察受监视元素的属性值变更。 默认为false

    characterData
        + 可选，设为true可以监视目标节点或子节点树中节点所包含的字符数据的变化，无默认值。

    characterDataOldValue
        + 可选，设为true在监听目标节点或子节点树中节点所包含的字符字符数据的原始数据，无默认值

    childList
        + 可选，设为true监视目标节点(如果subtree为true则包含子孙节点)添加或删除新的子节点，默认为false

    subtree
        + 可选，设为true以扩展监视范围到目标节点的整个子树的所有节点，MutationObserverInit的其他值都会作用于此子树下的所有节点，默认为false
