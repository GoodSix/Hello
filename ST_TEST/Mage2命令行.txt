# admin删除或解锁管理员

    admin:user:create
        + 创建一个管理员

    admin:user:unlock
        + 解锁管理员帐户

# app应用管理

    app:config:dump
        + 创建应用程序转储

    app:config:import
        + 将数据从共享配置文件导入适当的数据存储

    app:config:status
        + 检查配置传播是否需要更新

    config:sensitive:set
        + 设置敏感的配置值

# setup系统管理

    setup:backup
        + 备份Magento应用程序代码库，媒体和数据库

    setup:config:set
        + 创建或修改部署配置

    setup:cron:run
        + 运行计划用于安装应用程序的cron作业

    setup:db-data:upgrade
        + 在数据库中安装和升级数据

    setup:db-schema:upgrade
        + 安装和升级数据库架构

    setup:db:status
        + 检查数据库架构或数据是否需要升级

    setup:di:compile
        + 生成DI配置以及可以自动生成的所有缺少的类

    setup:install
        + 安装Magento应用程序

    setup:performance:generate-fixtures
        + 产生固定装置

    setup:rollback
        + 回滚Magento应用程序代码库，媒体和数据库

    setup:static-content:deploy
        + 部署静态视图文件

    setup:store-config:set
        + 安装商店配置。 从2.2.0版开始不推荐使用。 使用config：set代替

    setup:uninstall
        + 卸载Magento应用程序

    setup:upgrade
        + 升级Magento应用程序，数据库数据和架构
