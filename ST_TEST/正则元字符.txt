# 匹配范围
    \d
        + 匹配任意一个十进制数字
        + 等价于 [0-9]

    \D
        + 匹配除十进制数字以外的任意数字
        + 等价于 [^0-9]

    \s
        + 匹配空白字符
        + 等价于 [\n\f\r\t\v]

    \S
        + 匹配非空白字符
        + 等价于 [^\n\f\r\t\v]

    \w
        + 匹配任意一个数字、字符和下划线
        + 等价于 [0-9a-zA-Z_]

    \W
        + 匹配除字符、数字和下划线以外的任意字符
        + 等价于 [^0-9a-zA-Z_]

    []
        + 用来表示范围
        + 也可以匹配任意一个中括号中定义的原子

    [^]
        + 抑扬符表示排除


# 限定次数
    
    *
        + 匹配0次或多次其前的原子
        + 等价于 {0,}

    +
        + 匹配1次或多次其前的原子
        + 等价于 {1,}

    ?
        + 匹配0次或1次其前的原子
        + 等价于 {0,1}

    {n}
        + 表示其前的原子正好出现n次

    {n,}
        + 表示其前的原子至少出现n次，最多不限制

    {n,m}
        + 表示其前的原子至少出现n次，最多出现m次

# 其他

    .
        + 匹配除了换行符(\n)以外的任意字符[在Windows下还匹配\f\r]

    |
        + 两个或多个分支选择【优先级最低】

    ^
        + 匹配输入字符的开始位置

    $
        + 匹配输入字符的结束位置

    \b
        + 匹配词边界

    \B
        + 匹配非词边界

    ()
        + 模式单元，把多个小原子组成一个大院子。
        + 还能改变优先级
